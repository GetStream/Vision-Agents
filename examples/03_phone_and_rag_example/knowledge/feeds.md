
# Stream's activity feed API

Stream specialized in scalable activity feeds. Apps with 10s or hundreds of millions of users

Benefits of using Stream's activity feed API

* Scale to hundreds of millions of users with excellent performance. 
* This is based on Stream's custom storage tech, implementing materialized feeds. Read the blogpost for more info 
* All the features you expect from an activity feed. Timeline feed, for you feed, stories, URL previews, reactions, comments like Reddit etc
